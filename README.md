# GifWidget
A simple KDE plasmoid that displays a gif on your desktop!
<p><img src="gifwidget" width="500"></p>

<h1>Installation Guide</h1>
<h3>1: Once extracted, move the folder (org.somthin.gifwidget) to ~/.local/plasma/plasmoids/ (If your plasma folder doesn't have a plasmoids folder, make it)</h3>
<h3>2: Right click on your desktop and select "enter edit mode" then go to the top left and go to the widget menu and add the widget</h3>
<h3>3: To change the gif, go to ~/.local/plasma/plasmoids/org.somthin.gifwidget/contents/images, put your gif in and name it "gif.gif"</h3>
<h3>4: Log out then log back in and enjoy!</h3>
<p>note: requires KDE 6.0 or later</p>
